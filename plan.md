# Cafe Aroma Project Plan

## Goal
Make a cafe ordering app named "Cafe Aroma" with a desktop admin view and mobile-first client UI. Implement a single file for themes and fix existing errors.

## Branding
- **Name**: Cafe Aroma
- **Colors**: Light Cream background, Dark Brown accents.
- **Typography**: "Caveat" (Cursive, soothing, yet professional).

## Technical Requirements
- **Theme**: Centralized CSS variables in `globals.css`.
- **Responsive UI**: Fix mobile card descriptions with an expandable "dropdown" logic.
- **Authentication**: Fix Google OAuth by using the correct public environment variable.
- **Errors**: Rectify any identified code errors.

## Tasks
1. Update `globals.css` with branding variables.
2. Integrate "Caveat" font in `layout.tsx`.
3. Update `page.tsx` with "Cafe Aroma" and new mobile card UI.
4. Fix Google Client ID in `login/page.tsx`.
5. Conduct final audit and verification.
